## 什么是iOS
- Core OS。mac OS操作系统是BSD Unix内核
- Core Service。核心服务层，对底层的封装。
- Media。ipad iPhone都是多媒体设备。
- Cocoa Touch。交互页面。

### 平台组成
- 工具：Xcode
- 语言：Swift
- Framework：Foundation, UIKit, Core Data等
- 设计模式：MVC

Swift方法参数比别的语言的方法参数多了一个形参，可以让代码像句子一样，直接表达意思
```
let number = 0
flipCard(withEmoji: "🐢", on: sender)
```
